Kezdőoldal A könyvem

Gulácsi András vagyok. Full stack Javascript webfejlesztőnek és UX designer-nek tanulok. Eredeti végzettségemet tekintve geográfus és környezettudós vagyok. A szabadidőmben táncolok.

A fő képességeim: Node.js, Express keretrendszer, Jekyll, egyedi webdesign, GIT verziókezelő, npm csomagkezelő.

Elsajátításra váró ismeretek: MySQL, MongoDB és PostgreSQL adatbázisok.

Az állásajánlatokkal kapcsolatos megkereséseket az e-mailemre várom (friss önéletrajz).

npm csomagok
